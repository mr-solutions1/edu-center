# دليل التشغيل المحلي - أكاديمية ركان (Rakan Academy)

اتباع الخطوات التالية سيمكنك من تشغيل النظام بالكامل على جهازك الشخصي واختباره.

## 📋 المتطلبات المسبقة
- تثبيت **Node.js** (إصدار 20 أو أحدث).
- حساب **MongoDB Atlas** (أو قاعدة بيانات MongoDB تعمل محلياً).

---

## 1️⃣ إعداد قاعدة البيانات والـ API (Backend)

1. افتح نافذة "Terminal" (أو Command Prompt) وانتقل لمجلد الـ API:
   ```bash
   cd edu-core-api
   ```
2. قم بتثبيت المكتبات اللازمة:
   ```bash
   npm install
   ```
3. قم بإنشاء ملف الإعدادات:
   - ابحث عن الملف المسمى `.env.local.example`.
   - قم بتغيير اسمه إلى `.env`.
   - افتح الملف وقم بوضع رابط قاعدة البيانات الخاص بك (Atlas URI) في متغير `MONGO_URI`.

4. **إنشاء حساب المدير (Admin) الأول:**
   هذه الخطوة ضرورية لتتمكن من تسجيل الدخول:
   ```bash
   node create-admin.js
   ```
   *اتبع التعليمات التي ستظهر في الشاشة لإدخال بريدك الإلكتروني وكلمة المرور.*

5. تشغيل الـ API:
   ```bash
   npm run dev
   ```
   *سيظهر لك رسالة: `Server running on port 5000`.*

---

## 2️⃣ إعداد الواجهة الأمامية (Frontend)

1. افتح نافذة "Terminal" **جديدة** وانتقل لمجلد الـ Web:
   ```bash
   cd edu-core-web
   ```
2. قم بتثبيت المكتبات اللازمة:
   ```bash
   npm install
   ```
3. قم بإنشاء ملف الإعدادات:
   - ابحث عن الملف المسمى `.env.local.example`.
   - قم بتغيير اسمه إلى `.env`.
   - (الملف معد مسبقاً ليشير إلى `http://localhost:5000/api`).

4. تشغيل الواجهة:
   ```bash
   npm run dev
   ```
   *سيظهر لك رابط، عادة ما يكون: `http://localhost:5173`.*

---

## 3️⃣ الدخول واختبار النظام

1. افتح المتصفح وانتقل إلى الرابط: `http://localhost:5173`.
2. ستظهر لك صفحة تسجيل الدخول الخاصة بأكاديمية ركان.
3. استخدم **البريد الإلكتروني** و**كلمة المرور** اللذين أنشأتهما في الخطوة رقم (1.4).
4. بمجرد الدخول، ستتمكن من تصفح لوحة التحكم، إضافة طلاب، مدرسين، وجدولة الحصص.

---

### 💡 ملاحظات هامة:
- تأكد أن الـ API يعمل دائماً قبل محاولة تسجيل الدخول من المتصفح.
- في حال واجهت مشكلة في الاتصال بقاعدة البيانات، تأكد من إضافة IP جهازك في إعدادات (Network Access) في حساب MongoDB Atlas.
